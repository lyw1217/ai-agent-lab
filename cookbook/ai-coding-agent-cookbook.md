# AI Coding Agent Cookbook

스터디가 끝나면 이 문서에 팀에서 다시 쓸 수 있는 실전 workflow를 정리합니다.

## 목차 초안

1. Claude Code 실전 사용법
2. Codex 실전 사용법
3. Antigravity 실전 사용법
4. 상황별 추천 도구
5. 자주 쓰는 프롬프트
6. 자주 쓰는 slash command / hook / skill
7. 테스트 생성 workflow
8. 리팩터링 workflow
9. PR review workflow
10. 회사 업무에 적용할 때 주의사항

## 정리 원칙

- 실제로 써본 workflow만 남긴다.
- 성공 사례와 실패 사례를 함께 적는다.
- 팀원이 그대로 따라 할 수 있게 명령어와 prompt를 포함한다.
- 민감 정보나 회사 내부 정보는 제외한다.
