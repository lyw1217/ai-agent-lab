# AI Generated Code Review Policy

AI가 만든 코드는 반드시 사람이 검토합니다.

## 리뷰 체크리스트

- 요구사항을 실제로 만족하는가
- 관련 없는 변경이 섞이지 않았는가
- edge case와 오류 처리가 충분한가
- 테스트가 의미 있는 동작을 검증하는가
- 보안, 성능, 호환성 문제가 없는가
- 기존 코드 스타일과 architecture를 존중하는가
- 문서나 사용법 변경이 필요한가

## 검증 기준

- 가능한 경우 테스트를 실행한다.
- 테스트가 없다면 수동 검증 절차를 기록한다.
- agent의 설명만 믿지 않고 diff를 직접 읽는다.
